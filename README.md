# PPLA_project

Wstępny projekt aplikacji edukacyjnej dla pilotów PPLA, przygotowany w C# z wykorzystaniem Visual Studio.

## Opis
Aplikacja konsolowa umożliwiająca realizację podstawowych narzędzi i obliczeń lotniczych:

1. **Weight & Balance (W&B)** – oblicza masę startową, środek ciężkości i %MAC.
2. **Fuel & Flight Planner** – planowanie lotu, zużycie paliwa, czas w locie, paliwo blokowe.
3. **Runway Wind / Crosswind Calculator** – oblicza komponenty wiatru i ostrzega przy przekroczeniu limitu.
4. **METAR/TAF Parser** – prosty parser ręcznie wprowadzonego METAR/TAF.
5. **VFR Planner** – oblicza heading, ground speed, czas lotu i wind correction angle dla odcinków VFR.
